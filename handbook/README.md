# Handbook

本目录是 Async Collaboration Handbook 的正文入口，供人类与 AI 共同阅读。

Handbook 的正文应描述当前采用的协作做法，并解释为什么这样做、适用边界是什么、有哪些示例，以及 Agent 可以如何审阅。平台落地说明也可以放在 Handbook 中，但必须和平台无关的协议正文隔离，不能新增协议里没有的义务。

## 当前内容

- [手册编写规范](writing-guide.md)：说明 Handbook 应如何编写、审阅和维护。
- [研究材料](research/)：保存会长期影响本项目设计的调查材料；其中内容不是规范正文。

## 阅读顺序

首次参与本项目时，建议先阅读仓库根目录的 [README](../README.md)，再阅读[手册编写规范](writing-guide.md)。需要理解 GitLab Handbook 对本项目的启发时，再阅读[研究材料](research/)。
